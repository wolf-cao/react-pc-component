## Badge(标记)

#### 属性

| 属性名 | 类型   | 默认值    | 说明     | 其他值                      |
| :----- | :----- | :-------- | :------- | :-------------------------- |
| size   | string | 'normal'  | 大小     | `large`, `mini`             |
| type   | string | 'primary' | 类型颜色 | `success`,`error`,`warning` |
| value  | string |           |          |

#### 例子

```
<Badge size="mini" type="error" value="99" />
```
